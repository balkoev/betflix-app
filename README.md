## Запуск проекта

1. Получить ключ на сайте https://kinopoiskapiunofficial.tech/ и добавить в `.env` файл.
2. npm install
3. npm run dev

### Технологии

1. Javascript
2. React
3. Redux Toolkit + query
4. MUI
